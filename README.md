# Scrapper

Vehicle data scraper for UMVVS Tanzania (https://umvvs.tra.go.tz/)

## Features
- Scrapes vehicle makes, models, years, etc.
- REST API endpoint for data access
- Ready for Render deployment

## Setup
```bash
npm install
node server.js
```
